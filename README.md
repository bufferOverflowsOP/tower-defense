# Tower Defense

C++ SFML tower-defense game.

## Requirements

- CMake
- C++17 compiler
- SFML 3.1.0

## Build

```sh
cmake -B build
cmake --build build
```

## Run

```sh
./build/bin/main
```

## Format

```sh
npm install
npm run format
clang-format -i src/*.{cpp,h}
```

See [STYLE.md](STYLE.md).
